### GitHub Stats Card
```markdown
![GitHub Stats](https://github-readme-stats.vercel.app/api?username=YOUR_USERNAME&show_icons=true&theme=tokyonight&hide_border=true&count_private=true&include_all_commits=true)
```

**Available Themes:**
- `dark`, `radical`, `merko`, `gruvbox`, `tokyonight`, `onedark`, `cobalt`, `synthwave`, `highcontrast`, `dracula`, `github_dark`, `nord`, `nightowl`, `material`

**Parameters:**
- `&show_icons=true` - Show icons
- `&hide_border=true` - Hide border
- `&count_private=true` - Count private repos
- `&include_all_commits=true` - Include all commits
- `&hide=stars,commits` - Hide specific stats

### Typing SVG Animation
```markdown
[![Typing SVG](https://readme-typing-svg.demolab.com?font=Fira+Code&weight=600&size=28&duration=3000&pause=1000&color=36BCF7&center=true&vCenter=true&repeat=true&width=600&lines=Line+1;Line+2;Line+3)](https://git.io/typing-svg)
```

**Parameters:**
- `font` - Font family
- `size` - Font size
- `duration` - Duration per character (ms)
- `pause` - Pause between lines (ms)
- `color` - Text color (hex)
- `lines` - Multiple lines separated by semicolon

## ðŸŽ¨ Badge Styles

All badges support these styles:
- `flat` - Default flat style
- `flat-square` - Flat with square edges
- `plastic` - Plastic style with gradient
- `for-the-badge` - Larger badges
- `social` - Social media style

Example:
```markdown
?style=for-the-badge
```

## ðŸ’¡ Pro Tips

1. **Consistency**: Use the same theme across all widgets for a cohesive look
2. **Balance**: Don't overcrowd your profile - keep it clean and readable
3. **Mobile-friendly**: Test how your profile looks on mobile devices
4. **Loading time**: Too many external widgets can slow down loading
5. **Updates**: Keep your profile updated with latest achievements
6. **Accessibility**: Use appropriate contrast ratios and alt text
